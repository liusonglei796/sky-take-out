## 二、阿里云 OSS 配置

```yaml
    access-key-id: your-access-key-id

    bucket-name: your-bucket-name
```

### 2.2 配置类

**文件**: `sky-server/src/main/java/com/sky/config/OssConfiguration.java`

```java
@Configuration
public class OssConfiguration {
    
    @Value("${sky.alioss.endpoint}")
    private String endpoint;
    
    @Value("${sky.alioss.access-key-id}")
    private String accessKeyId;
    
    @Value("${sky.alioss.access-key-secret}")
    private String accessKeySecret;
    
    @Value("${sky.alioss.bucket-name}")
    private String bucketName;
    
    @Bean
    public AliOssUtil aliOssUtil() {
        return new AliOssUtil(endpoint, accessKeyId, accessKeySecret, bucketName);
    }
}
```

## 三、文件上传接口

### 3.1 Controller

**文件**: `sky-server/src/main/java/com/sky/controller/admin/CommonControler.java`

```java
@PostMapping("/upload")
public Result<String> upload(MultipartFile file) {
    // 1. 获取原始文件名
    String originalFilename = file.getOriginalFilename();
    
    // 2. 截取文件扩展名
    String extension = originalFilename.substring(originalFilename.lastIndexOf("."));
    
    // 3. 生成新文件名（UUID）
    String objectName = UUID.randomUUID().toString() + extension;
    
    // 4. 上传到 OSS
    String filePath = aliOssUtil.upload(file.getBytes(), objectName);
    
    // 5. 返回文件访问路径
    return Result.success(filePath);
}
```

### 3.2 OSS 工具类

**文件**: `sky-common/src/main/java/com/sky/utils/AliOssUtil.java`

```java
public String upload(byte[] bytes, String objectName) {
    // 1. 创建 OSS 客户端
    OSS ossClient = new OSSClientBuilder().build(endpoint, accessKeyId, accessKeySecret);
    
    try {
        // 2. 上传文件
        ossClient.putObject(bucketName, objectName, new ByteArrayInputStream(bytes));
    } finally {
        // 3. 关闭客户端
        ossClient.shutdown();
    }
    
    // 4. 返回访问URL
    return "https://" + bucketName + "." + endpoint + "/" + objectName;
}
```

## 五、菜品 CRUD 完整流程

### 5.1 新增菜品（带口味）

**Controller**:
```java
@PostMapping
public Result save(@RequestBody DishDTO dishDTO) {
    dishService.savewithFlavor(dishDTO);
    
    // 清理 Redis 缓存
    String key = "dish_" + dishDTO.getCategoryId();
    redisTemplate.delete(key);
    
    return Result.success();
}
```

**Service**:
```java
public void savewithFlavor(DishDTO dishDTO) {
    // 1. 保存菜品基本信息
    Dish dish = new Dish();
    BeanUtils.copyProperties(dishDTO, dish);
    dishMapper.insert(dish);
    
    // 2. 保存菜品口味
    Long dishId = dish.getId();
    List<DishFlavor> flavors = dishDTO.getFlavors();
    if (flavors != null && !flavors.isEmpty()) {
        flavors.forEach(flavor -> {
            flavor.setDishId(dishId);
            dishFlavorMapper.insert(flavor);
        });
    }
}
```

**Service - 业务校验**:
```java
public void deleteBatch(List<Long> ids) {
    // 1. 检查菜品是否起售
    for (Long id : ids) {
        Dish dish = dishMapper.getById(id);
        if (dish.getStatus() == 1) {
            throw new BusinessException("菜品正在起售，不能删除");
        }
    }
    
    // 2. 删除菜品
    dishMapper.deleteByIds(ids);
    
    // 3. 删除关联的口味
    dishFlavorMapper.deleteByDishIds(ids);
}
```

## 六、缓存处理

### 6.1 为什么需要缓存？

- 菜品列表查询频繁
- 数据变化不实时
- 提升查询性能

### 6.2 缓存实现

**新增菜品时清理缓存**:
```java
// 清理对应分类的缓存
String key = "dish_" + dishDTO.getCategoryId();
redisTemplate.delete(key);
```

**查询时先查缓存**:
```java
public List<Dish> list(Long categoryId) {
    // 1. 先查 Redis
    String key = "dish_" + categoryId;
    List<Dish> dishes = (List<Dish>) redisTemplate.opsForValue().get(key);
    
    if (dishes != null) {
        return dishes;  // 缓存中有数据，直接返回
    }
    
    // 2. 缓存没有，查数据库
    dishes = dishMapper.list(categoryId);
    
    // 3. 存入 Redis
    redisTemplate.opsForValue().set(key, dishes);
    
    return dishes;
}
```

## 八、常见问题

### Q1: 文件上传大小限制？
Spring Boot 默认限制 1MB，可在配置中修改：
```yaml
spring.servlet.multipart.max-file-size: 10MB
spring.servlet.multipart.max-request-size: 20MB
```

### Q2: OSS 上传失败？
- 检查 AccessKey 是否正确
- 检查 Bucket 是否设置为公共读
- 检查 endpoint 是否正确

### Q3: 缓存没有生效？
- 检查 Redis 是否启动
- 检查 key 是否正确
