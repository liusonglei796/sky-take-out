```java
public BusinessDataVO getBusinessData(LocalDate begin, LocalDate end) {
    // 1. 查询营业额
    Double turnover = orderMapper.sumTurnover(begin, end);
    
    // 2. 查询有效订单数
    Integer validOrderCount = orderMapper.countValidOrder(begin, end);
    
    // 3. 计算完成率
    Integer orderCount = orderMapper.countOrder(begin, end);
    Double completionRate = validOrderCount.doubleValue() / orderCount;
    
    // 4. 计算平均客单价
    Double avgAmount = turnover / validOrderCount;
    
    // 5. 查询新增用户
    Integer newUsers = userMapper.countNewUsers(begin, end);
    
    return new BusinessDataVO(turnover, validOrderCount, completionRate, avgAmount, newUsers);
}
```

### 4.2 导出接口参考实现

```java
@GetMapping("/admin/order/export")
public void export(HttpServletResponse response) throws IOException {
    // 1. 查询近30天数据
    LocalDate begin = LocalDate.now().minusDays(30);
    LocalDate end = LocalDate.now().minusDays(1);
    
    // 2. 查询订单数据
    List<Map<String, Object>> orderList = orderMapper.getOrderList(begin, end);
    
    // 3. 创建工作簿
    XSSFWorkbook workbook = new XSSFWorkbook();
    XSSFSheet sheet = workbook.createSheet("订单统计");
    
    // 4. 写入表头
    Row headerRow = sheet.createRow(0);
    headerRow.createCell(0).setCellValue("日期");
    headerRow.createCell(1).setCellValue("订单数");
    headerRow.createCell(2).setCellValue("销售额");
    
    // 5. 写入数据
    int rowNum = 1;
    for (Map<String, Object> order : orderList) {
        Row row = sheet.createRow(rowNum++);
        row.createCell(0).setCellValue(order.get("date").toString());
        row.createCell(1).setCellValue(Integer.parseInt(order.get("count").toString()));
        row.createCell(2).setCellValue(Double.parseDouble(order.get("amount").toString()));
    }
    
    // 6. 输出文件
    response.setContentType("application/vnd.ms-excel");
    response.setHeader("Content-Disposition", "attachment;filename=report.xlsx");
    workbook.write(response.getOutputStream());
    workbook.close();
}
```

## 五、定时任务

### 5.1 订单超时自动取消（参考实现）

**任务类**: `sky-server/src/main/java/com/sky/task/OrderTask.java`

```java
@Component
public class OrderTask {
    
    /**
     * 处理超时订单
     */
    @Scheduled(cron = "0 0 * * * ?")  // 每小时执行
    public void processTimeoutOrder() {
        // 查询超时未支付的订单
        LocalDateTime timeout = LocalDateTime.now().minusMinutes(15);
        List<Orders> orders = orderMapper.getTimeoutOrders(timeout);
        
        // 取消订单
        orders.forEach(order -> {
            order.setStatus(OrderStatus.CANCELLED);
            order.setCancelReason("订单超时自动取消");
            order.setCancelTime(LocalDateTime.now());
            orderMapper.update(order);
        });
    }
}
```

### 5.2 启用定时任务

在启动类添加 `@EnableScheduling` 注解：

```java
@SpringBootApplication
@EnableScheduling
public class SkyApplication {
    public static void main(String[] args) {
        SpringApplication.run(SkyApplication.class, args);
    }
}
```

## 七、常见问题

### Q1: 导出文件很大？
- 使用流式写入
- 限制导出时间范围

### Q2: 日期格式错误？
- 检查时区设置
- 使用 LocalDateTime 而非 Date

### Q3: 定时任务不执行？
- 检查是否添加 `@EnableScheduling`
- 检查 cron 表达式
