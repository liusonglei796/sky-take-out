# 05 - 套餐管理 + 关联查询

> 本教程讲解套餐管理模块及 MyBatis 关联查询

## 一、套餐表设计

### 1.1 套餐表 + 套餐菜品关联表

```sql
-- 套餐表
CREATE TABLE `setmeal` (
  `id` bigint NOT NULL AUTO_INCREMENT,
  `category_id` bigint DEFAULT NULL COMMENT '分类id',
  `name` varchar(50) DEFAULT NULL COMMENT '套餐名称',
  `price` decimal(10,2) DEFAULT NULL COMMENT '套餐价格',
  `image` varchar(200) DEFAULT NULL COMMENT '图片',
  `description` varchar(200) DEFAULT NULL COMMENT '描述信息',
  `status` int DEFAULT '1' COMMENT '0停售 1起售',
  `create_time` datetime DEFAULT NULL,
  `update_time` datetime DEFAULT NULL,
  `create_user` bigint DEFAULT NULL,
  `update_user` bigint DEFAULT NULL,
  PRIMARY KEY (`id`)
);

-- 套餐菜品关联表
CREATE TABLE `setmeal_dish` (
  `id` bigint NOT NULL AUTO_INCREMENT,
  `setmeal_id` bigint DEFAULT NULL COMMENT '套餐id',
  `dish_id` bigint DEFAULT NULL COMMENT '菜品id',
  `name` varchar(50) DEFAULT NULL COMMENT '菜品名称',
  `price` decimal(10,2) DEFAULT NULL COMMENT '菜品单价',
  `copies` int DEFAULT NULL COMMENT '菜品份数',
  PRIMARY KEY (`id`)
);
```

### 2.3 Mapper 查询

**SetmealMapper.xml** - 关联查询:
```xml
<!-- 查询套餐及包含的菜品 -->
<select id="getByIdWithDish" resultMap="setmealVOMap">
    SELECT 
        s.*,
        d.id as dish_id,
        d.name as dish_name,
        d.price as dish_price,
        sd.copies
    FROM setmeal s
    LEFT JOIN setmeal_dish sd ON s.id = sd.setmeal_id
    LEFT JOIN dish d ON sd.dish_id = d.id
    WHERE s.id = #{id}
</select>

<resultMap id="setmealVOMap" type="SetmealVO">
    <id column="id" property="id"/>
    <result column="name" property="name"/>
    ...
    <!-- 一对多 collection -->
    <collection property="setmealDishes" ofType="SetmealDish">
        <id column="dish_id" property="dishId"/>
        <result column="dish_name" property="name"/>
        <result column="dish_price" property="price"/>
        <result column="copies" property="copies"/>
    </collection>
</resultMap>
```

## 三、套餐 CRUD

### 3.1 新增套餐（带菜品）

**Controller**:

```java
    setmealService.saveWithDish(setmealDTO);
    return Result.success();

```

**Service**:
```java
public void saveWithDish(SetmealDTO setmealDTO) {
    // 1. 保存套餐基本信息
    Setmeal setmeal = new Setmeal();
    BeanUtils.copyProperties(setmealDTO, setmeal);
    setmealMapper.insert(setmeal);
    
    // 2. 保存套餐菜品关联
    Long setmealId = setmeal.getId();
    List<SetmealDish> dishes = setmealDTO.getSetmealDishes();
    
    if (dishes != null && !dishes.isEmpty()) {
        dishes.forEach(dish -> {
            dish.setSetmealId(setmealId);
            setmealDishMapper.insert(dish);
        });
    }
}
```

### 4.2 使用注解缓存

```java
// 新增套餐后清理缓存
@CacheEvict(cacheNames="setmealCache", key="#setmealDTO.categoryId")
public void saveWithDish(SetmealDTO setmealDTO) { ... }

// 删除时清理所有套餐缓存
@CacheEvict(cacheNames="setmealCache", allEntries = true)
public void deleteBatch(List<Long> ids) { ... }

// 查询时使用缓存
@Cacheable(cacheNames="setmealCache", key="#categoryId")
public List<Setmeal> list(Long categoryId) { ... }
```

### 4.3 缓存注解说明

| 注解 | 作用 |
|------|------|
| `@Cacheable` | 查询时缓存 |
| `@CacheEvict` | 删除/更新时清除缓存 |
| `@CachePut` | 更新后写入缓存 |

## 六、常见问题

### Q1: 套餐关联的菜品如何展示？
使用 `LEFT JOIN` 关联查询，在 VO 中用 `List<SetmealDish>` 接收

### Q2: 缓存没有生效？
检查是否添加了 `@EnableCaching` 注解

### Q3: 删除套餐需要同时删除关联？
是的，需要在事务中删除 `setmeal_dish` 表中的关联数据
