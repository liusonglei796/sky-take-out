# 06 - 购物车 + Redis 缓存实战

> 本教程讲解购物车模块及 Redis 缓存的实战应用

## 一、购物车模块概述

### 1.1 功能特点
- 用户端功能（需要用户登录）
- 临时存储用户的菜品/套餐选择
- 支持菜品口味选择
- 数据存储在 Redis 中

### 1.2 购物车表设计

```sql
-- 购物车表（实际存储在 Redis）
CREATE TABLE `shopping_cart` (
  `id` bigint NOT NULL AUTO_INCREMENT,
  `name` varchar(50) DEFAULT NULL COMMENT '商品名称',
  `image` varchar(200) DEFAULT NULL COMMENT '图片',
  `user_id` bigint DEFAULT NULL COMMENT '用户id',
  `dish_id` bigint DEFAULT NULL COMMENT '菜品id',
  `setmeal_id` bigint DEFAULT NULL COMMENT '套餐id',
  `dish_flavor` varchar(50) DEFAULT NULL COMMENT '菜品口味',
  `number` int DEFAULT '1' COMMENT '数量',
  `amount` decimal(10,2) DEFAULT NULL COMMENT '金额',
  `create_time` datetime DEFAULT NULL COMMENT '创建时间',
  PRIMARY KEY (`id`)
);
```

## 三、购物车接口

### 3.1 添加购物车

**Controller**:
```java
@PostMapping("/add")
public Result add(@RequestBody ShoppingCartDTO shoppingCartDTO) {
    shoppingCartService.add(shoppingCartDTO);
    return Result.success();
}
```

**Service**:
```java
public void add(ShoppingCartDTO dto) {
    // 1. 获取当前用户
    Long userId = BaseContext.getCurrentId();
    
    // 2. 构建购物车对象
    ShoppingCart cart = new ShoppingCart();
    cart.setUserId(userId);
    cart.setNumber(1);
    cart.setCreateTime(LocalDateTime.now());
    
    // 3. 判断是菜品还是套餐
    if (dto.getDishId() != null) {
        // 菜品
        Dish dish = dishMapper.getById(dto.getDishId());
        cart.setName(dish.getName());
        cart.setImage(dish.getImage());
        cart.setAmount(dish.getPrice());
        cart.setDishId(dish.getDishId());
    } else {
        // 套餐
        Setmeal setmeal = setmealMapper.getById(dto.getSetmealId());
        cart.setName(setmeal.getName());
        cart.setImage(setmeal.getImage());
        cart.setAmount(setmeal.getPrice());
        cart.setSetmealId(dto.getSetmealId());
    }
    
    // 4. 检查购物车是否已存在
    ShoppingCart existing = mapper.findByUserIdAndDishId(cart);
    if (existing != null) {
        // 数量+1
        existing.setNumber(existing.getNumber() + 1);
        mapper.updateNumber(existing);
    } else {
        // 新增
        mapper.insert(cart);
    }
}
```

### 3.2 查看购物车

```java
@GetMapping("/list")
public Result<List<ShoppingCart>> list() {
    Long userId = BaseContext.getCurrentId();
    List<ShoppingCart> list = shoppingCartMapper.list(userId);
```

### 3.3 清空购物车

```java
@DeleteMapping("/clean")
public Result clean() {
    Long userId = BaseContext.getCurrentId();
    mapper.deleteByUserId(userId);
    return Result.success();
}
```

## 五、Redis 缓存策略

### 5.1 为什么用 Redis？

| 特点 | 说明 |
|------|------|
| 性能高 | 内存存储，读写快 |
| 支持复杂数据结构 | List/Hash 都能用 |
| 天然支持过期 | 可设置购物车过期时间 |

### 5.2 缓存操作示例

```java
@Autowired
private RedisTemplate redisTemplate;

// 存储
public void addToCart(ShoppingCart cart) {
    String key = "cart:" + cart.getUserId();
    List<ShoppingCart> list = redisTemplate.opsForList().range(key, 0, -1);
    list.add(cart);
    redisTemplate.opsForList().leftPush(key, list);
}

// 读取
public List<ShoppingCart> getCart(Long userId) {
    String key = "cart:" + userId;
    return redisTemplate.opsForList().range(key, 0, -1);
}

// 清空
public void clean(Long userId) {
    String key = "cart:" + userId;
    redisTemplate.delete(key);
}
```

## 六、常见问题

### Q1: 购物车数据存在哪里？
Redis 中，以 `cart:userId` 为 key

### Q2: 如何区分菜品和套餐？
通过 `dishId` 和 `setmealId` 字段判断，两者只能有一个有值

### Q3: 购物车需要持久化吗？
不需要，用户退出登录后购物车数据可以清除或保留一定时间
