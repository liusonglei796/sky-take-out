## 二、支付实现

### 2.1 统一下单

```java
public OrderPaymentVO payment(OrdersPaymentDTO dto) throws Exception {
    // 1. 查询订单
    Orders order = orderMapper.getByNumber(dto.getOrderNumber());
    
    // 2. 调用微信支付统一下单
    HttpResponse response = WeChatPayUtil.unifiedOrder(
        order.getId().toString(),
        order.getAmount().multiply(new BigDecimal(100)).longValue(),
        "苍穹外卖订单",
        "NATIVE"
    );
    
    // 3. 解析返回的支付参数
    JSONObject jsonObject = JSONObject.parseObject(response.getContent());
    String codeUrl = jsonObject.getString("code_url");
    
    return new OrderPaymentVO(codeUrl);
}
```

### 2.2 支付回调

```java
@PostMapping("/notify")
public Result notify(HttpServletRequest request) {
    // 1. 读取回调数据
    String xmlData = readXml(request);
    
    // 2. 验签
    if (!WeChatPayUtil.verifySign(xmlData)) {
        return Result.error("验签失败");
    }
    
    // 3. 更新订单状态
    String orderNumber = extractOrderNumber(xmlData);
    orderService.paySuccess(orderNumber);
    
    // 4. 返回成功
    return Result.success("<xml><return_code><![CDATA[SUCCESS]]></return_code></xml>");
}
```

### 2.3 支付成功处理

```java
public void paySuccess(String orderNumber) {
    // 1. 更新订单状态
    Orders order = orderMapper.getByNumber(orderNumber);
    order.setStatus(OrderStatus.PENDING_RECEIPT); // 待接单
    order.setPayStatus(1); // 已支付
    order.setCheckoutTime(LocalDateTime.now());
    orderMapper.update(order);
}
```

## 五、常见问题

### Q1: 支付回调收不到？
- 检查外网映射（内网穿透）
- 检查防火墙
- 检查回调地址是否正确

### Q2: 验签失败？
- 检查密钥是否正确
- 检查参数顺序

### Q3: 订单金额为0？
- 金额单位是分，需要 * 100
