# 01 - 环境搭建与项目启动

> 本教程将帮助你搭建开发环境并成功启动项目

## 一、环境要求

| 软件 | 版本要求 | 说明 |
|------|----------|------|
| JDK | 25 | 项目使用 Spring Boot 4.0 + Java 25 |
| Maven | 3.9+ | 构建工具 |
| MySQL | 8.0+ | 数据库 |
| Redis | 7.0+ | 缓存 |
| Docker | 24+ | 容器化部署（可选） |
| IntelliJ IDEA | 2024+ | 推荐 IDE |

## 二、项目结构

```
sky-take-out/
├── sky-common/          # 公共模块（工具类、配置）
├── sky-pojo/           # 实体类（Entity/DTO/VO）
├── sky-server/         # 核心业务模块
│   └── src/main/
│       ├── java/com/sky/
│       │   ├── controller/  # 控制层
│       │   ├── service/     # 业务层
│       │   ├── mapper/      # 数据层
│       │   └── config/      # 配置类
│       └── resources/
│           ├── application.yml
│           └── mapper/       # MyBatis XML
└── docs/
    └── tutorial/       # 教程目录
```

## 三、配置步骤

### 3.1 导入项目

1. 打开 IDEA，选择 `File` → `Open` → 选择项目根目录
2. 等待 Maven 自动下载依赖（首次约 5-10 分钟）
3. 确保 JDK 选择正确：`File` → `Project Structure` → `Project` → `SDK`

### 3.2 配置数据库

**方式一：使用 Docker（推荐）**

```bash
# 启动 MySQL
docker run -d --name sky-mysql \
  -p 3306:3306 \
  -e MYSQL_ROOT_PASSWORD=root123 \
  -e MYSQL_DATABASE=sky_take_out \
  mysql:8.0

# 启动 Redis
docker run -d --name sky-redis \
  -p 6379:6379 \
  redis:7-alpine
```

**方式二：本地安装**

- 下载 MySQL 8.0：https://dev.mysql.com/downloads/mysql/
- 下载 Redis：https://redis.io/download/

### 3.3 导入数据库

> 注意：需要获取数据库初始化脚本（联系项目作者或自行根据 Entity 生成）

```bash
# 登录 MySQL
mysql -u root -p

# 执行 SQL 脚本
source /path/to/sky_take_out.sql
```

### 3.4 修改配置文件

编辑 `sky-server/src/main/resources/application.yml`，添加数据源和Redis配置：

```yaml
sky:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    host: localhost
    port: 3306
    database: sky_take_out
    username: root
    password: root123  # 改成你的密码

  redis:
    host: localhost
    port: 6379
    # password: 如果有密码则填写
```

## 四、启动项目

### 4.1 方式一：Docker 一键部署（推荐）

```bash
# 克隆项目后，进入目录
cd sky-take-out

# 一键启动所有服务（MySQL + Redis + 应用）
docker-compose up -d

# 查看日志
docker-compose logs -f app
```

访问地址：http://localhost:8081/doc.html

### 4.2 方式二：IDEA 启动

1. 找到 `sky-server/src/main/java/com/sky/SkyApplication.java`
2. 右键 → `Run 'SkyApplication'`
3. 等待启动，看到以下日志表示成功：

```
Started SkyApplication in X.XXX seconds
```

### 4.3 方式三：命令行启动

```bash
# 打包项目
mvn clean package -DskipTests

# 运行
java -jar sky-server/target/sky-server-1.0-SNAPSHOT.jar
```

## 六、常见问题

### Q1: 端口被占用
```bash
# 查看端口占用
netstat -ano | findstr 8080

# 结束进程
taskkill /PID <PID> /F
```

### Q2: Maven 下载依赖慢
修改 Maven 镜像源：

```xml
<!-- maven/conf/settings.xml -->
<mirrors>
  <mirror>
    <id>aliyun</id>
    <mirrorOf>*</mirrorOf>
    <url>https://maven.aliyun.com/repository/public</url>
  </mirror>
</mirrors>
```

### Q3: 中文乱码
在 IDEA 中：`File` → `Settings` → `Editor` → `File Encodings`
- Project Encoding: UTF-8
- Default encoding for properties files: UTF-8
