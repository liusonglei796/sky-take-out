# 苍穹外卖项目注解大全

> 本文档详细解释项目中使用的所有注解及其用法

## 一、Lombok 注解

Lombok 通过注解简化 Java 代码，自动生成 getter/setter/构造函数等。

### 1.1 @Data

```java
@Data
public class User {
    private Long id;
    private String name;
}
```

**作用**：自动生成 `equals()`、`hashCode()`、`toString()`、`getter`、`setter` 方法

**等价于**：
```java
public class User {
    private Long id;
    private String name;
    
    public Long getId() { return id; }
    public void setId(Long id) { this.id = id; }
    // ... 其他 getter/setter
    // ... equals, hashCode, toString
}
```

---

### 1.2 @Builder

```java
@Builder
public class User {
    private Long id;
    private String name;
}
```

**作用**：启用建造者模式创建对象

**用法**：
```java
User user = User.builder()
    .id(1L)
    .name("张三")
    .build();
```

---

### 1.3 @NoArgsConstructor

```java
@NoArgsConstructor
public class User { }
```

**作用**：生成无参构造函数

---

### 1.4 @AllArgsConstructor

```java
@AllArgsConstructor
public class User {
    private Long id;
    private String name;
}
```

**作用**：生成全参构造函数

```java
User user = new User(1L, "张三");
```

## 二、Spring 核心注解

### 2.1 @Component

```java
@Component
public class JwtTokenInterceptor { }
```

**作用**：将类注册为 Spring Bean，相当于 `<bean>` 配置

**衍生注解**（功能相同，语义不同）：
- `@Service` - 业务层
- `@Repository` - 数据访问层
- `@Controller` / `@RestController` - 控制层

---

### 2.2 @Service

```java
@Service
public class UserService { }
```

**作用**：标注业务层组件，Spring 会自动扫描并注入

---

### 2.3 @RestController

```java
@RestController
@RequestMapping("/user")
public class UserController { }
```

**作用**：
- `@Controller` + `@ResponseBody`
- 返回 JSON/XML 数据，而非视图

---

### 2.4 @RequestMapping

```java
@RequestMapping("/admin/user")
public class UserController { }
```

**作用**：映射 URL 路径

**衍生注解**：
- `@GetMapping` - GET 请求
- `@PostMapping` - POST 请求
- `@PutMapping` - PUT 请求
- `@DeleteMapping` - DELETE 请求

---

### 2.5 @Autowired

```java
@Autowired
private UserService userService;
```

**作用**：自动注入 Bean（byType）

**注意**：Spring 4.3+ 构造函数注入更推荐

```java
// 推荐方式
public class UserController {
    private final UserService userService;
    
    public UserController(UserService userService) {
        this.userService = userService;
    }
}
```

---

### 2.6 @Configuration

```java
@Configuration
public class RedisConfiguration {
    @Bean
    public RedisTemplate redisTemplate() {
        return new RedisTemplate();
    }
}
```

**作用**：标注配置类，相当于 Spring XML 配置文件

---

### 2.7 @Bean

```java
@Bean
public RedisTemplate redisTemplate() {
    return new RedisTemplate();
}
```

**作用**：在配置类中声明 Bean

---

### 2.8 @Value

```java
@Value("${spring.datasource.url}")
private String url;
```

**作用**：注入配置文件中的值

---

### 2.9 @Transactional

```java
@Transactional
public void saveOrder() {
    // 多个数据库操作，要么全部成功，要么全部回滚
}
```

**作用**：开启事务管理

**常用参数**：
| 参数 | 说明 |
|------|------|
| `rollbackFor` | 指定回滚的异常类型 |
| `isolation` | 事务隔离级别 |
| `timeout` | 超时时间 |

---

### 2.10 @Slf4j

```java
@Slf4j
public class UserService {
    public void test() {
        log.info("日志信息");
        log.error("错误信息");
    }
}
```

**作用**：自动生成日志对象 `log`

**底层实现**：
```java
private static final org.slf4j.Logger log = org.slf4j.LoggerFactory.getLogger(UserService.class);
```

---

### 2.11 @SpringBootApplication

```java
@SpringBootApplication
public class SkyApplication {
    public static void main(String[] args) {
        SpringApplication.run(SkyApplication.class, args);
    }
}
```

**作用**：组合注解，包含：
- `@Configuration` - 配置类
- `@EnableAutoConfiguration` - 启用自动配置
- `@ComponentScan` - 组件扫描

---

### 2.12 @EnableTransactionManagement

```java
@EnableTransactionManagement
public class SkyApplication { }
```

**作用**：启用注解方式的事务管理

---

### 2.13 @PathVariable

```java
@GetMapping("/user/{id}")
public User getUser(@PathVariable Long id) {
    return userService.getById(id);
}
```

**作用**：获取 URL 路径中的参数

---

### 2.14 @RequestBody

```java
@PostMapping("/user")
public Result save(@RequestBody UserDTO userDTO) {
    return Result.success();
}
```

**作用**：将请求体 JSON 转换为 Java 对象

---

### 2.15 @RequestParam

```java
@GetMapping("/user/list")
public List<User> list(@RequestParam(defaultValue = "1") Integer page,
                       @RequestParam(required = false) String name) {
    return userService.list(page, name);
}
```

**作用**：获取请求参数

---

### 2.16 @RestControllerAdvice

```java
@RestControllerAdvice
public class GlobalExceptionHandler {
    @ExceptionHandler(Exception.class)
    public Result handleException(Exception e) {
        return Result.error(e.getMessage());
    }
}
```

**作用**：全局异常处理，相当于 `ControllerAdvice` + `ResponseBody`

---

### 2.17 @ExceptionHandler

```java
@ExceptionHandler(BusinessException.class)
public Result handleBusinessException(BusinessException e) {
    return Result.error(e.getMessage());
}
```

**作用**：指定异常处理方法

---

### 2.18 @ResponseStatus

```java
@ExceptionHandler(BusinessException.class)
@ResponseStatus(HttpStatus.OK)
public Result handleException(Exception e) { }
```

**作用**：指定 HTTP 响应状态码

## 三、MyBatis 注解

### 3.1 @Mapper

```java
@Mapper
public interface UserMapper {
    @Select("SELECT * FROM user WHERE id = #{id}")
    User getById(Long id);
}
```

**作用**：声明为 MyBatis Mapper 接口

---

### 3.2 @Select / @Insert / @Update / @Delete

```java
@Mapper
public interface UserMapper {
    @Select("SELECT * FROM user WHERE id = #{id}")
    User getById(Long id);
    
    @Insert("INSERT INTO user(name) VALUES(#{name})")
    @Options(useGeneratedKeys = true, keyProperty = "id")
    int insert(User user);
    
    @Update("UPDATE user SET name=#{name} WHERE id=#{id}")
    int update(User user);
    
    @Delete("DELETE FROM user WHERE id = #{id}")
    int deleteById(Long id);
}
```

**作用**：声明 SQL 语句

---

### 3.3 @Param

```java
@Select("SELECT * FROM user WHERE name LIKE CONCAT('%', #{name}, '%')")
List<User> searchByName(@Param("name") String name);
```

**作用**：给参数指定别名，在 SQL 中使用

---

### 3.4 @Options

```java
@Insert("INSERT INTO user(name) VALUES(#{name})")
@Options(useGeneratedKeys = true, keyProperty = "id")
int insert(User user);
```

**作用**：配置插入操作选项

| 参数 | 说明 |
|------|------|
| `useGeneratedKeys` | 是否返回自增主键 |
| `keyProperty` | 主键属性名 |

## 四、数据校验注解

### 4.1 @Valid

```java
@PostMapping("/user")
public Result save(@Valid @RequestBody UserDTO userDTO) {
    return Result.success();
}
```

**作用**：触发实体类中的校验注解

---

### 4.2 @NotNull / @NotBlank / @NotEmpty

```java
@Data
public class UserDTO {
    @NotBlank(message = "用户名不能为空")
    private String username;
    
    @NotNull(message = "年龄不能为空")
    private Integer age;
    
    @NotEmpty(message = "列表不能为空")
    private List<String> tags;
}
```

| 注解 | 作用 |
|------|------|
| `@NotNull` | 不能为 null |
| `@NotBlank` | 不能为空字符串（仅 String） |
| `@NotEmpty` | 不能为空（Collection/Map/数组） |

---

### 4.3 @Size

```java
@Size(min = 2, max = 32, message = "名称长度在 2-32 之间")
private String name;

@Size(max = 255, message = "描述不能超过 255 字符")
private String description;
```

**作用**：限制字符串/集合大小

---

### 4.4 @Min / @Max

```java
@Min(value = 0, message = "值不能小于 0")
@Max(value = 1, message = "值不能大于 1")
private Integer status;
```

**作用**：限制数值范围

---

### 4.5 @Pattern

```java
@Pattern(regexp = "^1[3-9]\\d{9}$", message = "手机号格式不正确")
private String phone;

@Pattern(regexp = "^\\d{17}[\\dXx]$", message = "身份证格式不正确")
private String idNumber;
```

**作用**：正则表达式校验

---

### 4.6 @DecimalMin / @DecimalMax

```java
@DecimalMin(value = "0.01", message = "价格必须大于 0")
@DecimalMax(value = "99999.99", message = "价格不能超过 99999.99")
private BigDecimal price;
```

**作用**：限制小数精度

---

## 六、自定义注解

### 6.1 @AutoFill

**位置**：`sky-server/src/main/java/com/sky/annotation/AutoFill.java`

```java
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface AutoFill {
    OperationType value();
}
```

**作用**：用于标识需要自动填充公共字段的方法（如 createTime、updateTime）

**使用示例**：
```java
@AutoFill(OperationType.INSERT)
void insert(Category category);

@AutoFill(OperationType.UPDATE)
void update(Category category);
```

---

### 6.2 @Pointcut / @Before / @Aspect

**位置**：`sky-server/src/main/java/com/sky/aspect/AutoFillAspect.java`

```java
@Aspect
@Component
@Slf4j
public class AutoFillAspect {
    
    // 切入点：拦截所有标记了 @AutoFill 注解的 mapper 方法
    @Pointcut("execution(* com.sky.mapper.*.*(..)) && @annotation(com.sky.annotation.AutoFill)")
    public void autoFillPointCut() { }
    
    // 前置通知：在方法执行前执行
    @Before("autoFillPointCut()")
    public void autoFill(JoinPoint joinPoint) {
        // 自动填充逻辑
    }
}
```

| 注解 | 说明 |
|------|------|
| `@Aspect` | 声明为切面类 |
| `@Pointcut` | 定义切入点表达式 |
| `@Before` | 前置通知 |
| `@After` | 后置通知 |
| `@AfterReturning` | 返回通知 |
| `@AfterThrowing` | 异常通知 |
| `@Around` | 环绕通知 |

### 7.1 @JsonFormat

### 7.3 @SuppressWarnings

```java
@SuppressWarnings("unchecked")

    return (List<T>) cache.get(key);

```

**作用**：抑制编译警告

| 值 | 说明 |
|-----|------|
| `unchecked` | 未检查的转换 |
| `deprecation` | 已废弃 API |
| `all` | 所有警告 |

### 7.4 @ConditionalOnMissingBean

```java

    return new RedisTemplate();
}
```

**作用**：当容器中不存在指定 Bean 时才创建

---

### 7.5 @Override

```java
@Override

    // 重写方法
}
```

**作用**：标识重写父类/接口方法

### 7.6 @WebSocket 相关注解

```java
@ServerEndpoint("/ws/{sid}")
public class WebSocketServer {
    
    @OnOpen
    public void onOpen(Session session, @PathParam("sid") String sid) { }
    
    @OnMessage
    public void onMessage(String message, Session session) { }
    
    @OnClose
    public void onClose(Session session) { }
    
    @OnError
    public void onError(Session session, Throwable error) { }

```

| 注解 | 说明 |
|------|------|
| `@ServerEndpoint` | 声明 WebSocket 端点 |
| `@OnOpen` | 连接建立 |
| `@OnMessage` | 收到消息 |
| `@OnClose` | 连接关闭 |
| `@OnError` | 发生错误 |

## 九、常见问题

### Q1: @Data 和 @Builder 能一起用吗？
可以，Lombok 4.0+ 支持两者同时使用

### Q2: @Autowired 注入为 null？
检查：
1. Bean 是否被扫描到（@ComponentScan）
2. 是否有多个同类型 Bean（使用 @Qualifier）

### Q3: @Transactional 不生效？
检查：
1. 方法是否是 public
2. 异常是否被 catch 吞掉了
3. 是否有自调用（this.method()）

### Q4: @Valid 和 BindingResult 顺序？
```java
public Result save(@Valid @RequestBody UserDTO dto, BindingResult result) {
    if (result.hasErrors()) {
        return Result.error(result.getFieldError().getDefaultMessage());
    }
    // ...
}
```
